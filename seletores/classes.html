Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por TipoBusca elementos por um tag HTML especifica
  • Seletor por ID (#)Busca elementos através do atributo "id"
  • Seletor por classe (.)Busca elementos através do atributo "class"
  • Seletor Universal (*)Busca todos os elementos HTML
  • Seletor de Atributo ([atrib])Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
